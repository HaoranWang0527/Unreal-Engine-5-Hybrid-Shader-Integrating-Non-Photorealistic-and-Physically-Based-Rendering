# Unreal-Engine-5-Hybrid-Shader-Integrating-Non-Photorealistic-and-Physically-Based-Rendering
The aim of this project is to design and implement a hybrid shader in Unreal Engine 5 that seamlessly integrates Non-Photorealistic Rendering (NPR) techniques with Physically Based Rendering (PBR) material workflows. Like Arknights: Endfield
